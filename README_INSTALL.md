# Nexus Net - Network Visualization Platform

## Installation

Ce projet nécessite **Node.js**.

1.  Installez Node.js depuis [nodejs.org](https://nodejs.org/).
2.  Ouvrez un terminal dans ce dossier.
3.  Installez les dépendances :
    ```bash
    npm install
    ```

## Lancement

Pour lancer le serveur :

```bash
npm start
```

Ensuite, ouvrez votre navigateur sur : [http://localhost:3000](http://localhost:3000)

## Fonctionnalités

*   **Système de compte** : Inscription et Connexion sécurisée.
*   **Dashboard** : Gestion de vos plans réseaux.
*   **Éditeur** : L'outil Nexus Net complet avec sauvegarde automatique sur le serveur.
*   **Base de données** : SQLite (fichier `nexus.db` créé automatiquement).
