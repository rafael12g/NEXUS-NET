# Contributing

Thanks for considering contributing to this project! This project is **in development**, so changes may evolve quickly.

## Prerequisites
- Node.js (LTS recommended)
- npm

## Setup
1. Fork the repository and clone your fork.
2. Install dependencies:
   - `npm install`

## Run locally
- Start: `npm start` (or the command described in the README)

## Propose a contribution
1. Create a branch from `main`:
   - `git checkout -b feat/my-contribution`
2. Make your changes and test them.
3. Commit with a clear message (e.g., `feat: add user settings`).
4. Push the branch and open a Pull Request.

## Best practices
- Keep PRs small and focused.
- Describe the context and the solution.
- Update documentation when needed.

## Report a bug
- Open an issue with:
  - steps to reproduce
  - expected result
  - actual result
  - environment (OS, Node version)

## Security
For vulnerabilities, please contact the maintainers privately.
